# Installation in Linux:
**Open terminal and run:**
```shell script
 sudo apt update && apt full-upgrade
 ```
 ```shell script
  git clone https://github.com/arxhr007/Gamer-tux
```
```shell script
  cd Gamer-tux
```
```shell script
  bash install.sh
```
* Now Gamer-Tux is Succesfully installed in your system

## To uninstall in linux:
```shell script
  cd  Gamer-tux
```
```shell script
  bash uninstall.sh
```
* done!
